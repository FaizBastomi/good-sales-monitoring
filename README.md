# Good Sales Monitoring
This repository is used to fulfill **Data Structures and Algorithms** *Final Project*

## How to use
1. Clone it with Git
```sh
git clone https://github.com/FaizBastomi/good-sales-monitoring.git
```
2. Move to the cloned directory
```sh
cd good-sales-monitoring
```
3. Open with CodeBlocks (select the `.cbp` file)
4. Press `F9` to build and run
